# Meteológica — Prueba de selección para vacante de programador Frontend/Fullstack
Aplicación web que utiliza los datos del archivo data.yml para visualizar datos de predicción meteorológica.

Aplicación desarrollada con Svelte + Vite. 

¿Por qué Svelte?
- Es un framwork ligero, reactivo y muy eficiente para construir interfaces en tiempo real.
- Permite escribir menos código sin sacrificar rendimiento.
- Se compila a JS puro, haciendo que la app sea rápida y con un tamañao reducido.
- Fácil de desplegar.

# Caracterísiticas principales
- Visualización dinámica de temperatura y energía en gráficas interactivas (Chart.js).  
- Sincronización en tiempo real con la hora actual del navegador.  
- Actualización automática cada 5 segundos.  
- Arquitectura modular con componentes Svelte.  

# Instalación y ejecución
- Tener previamente instalado Node.js y npm.
- Copiar los archivos del proyecto
- Instalar dependencias: **npm install**
- Lanzar la aplicación: **npm run dev**
